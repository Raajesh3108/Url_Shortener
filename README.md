## URL Shortener application using NodeJs & MongoDB

URL shortening application that takes user input and generates a shortened URL as output.

**MongoDB** is used for storing Users in Database.

---

## To start setting up the project

Step 1: Clone the repository

```bash
git clone https://github.com/Raajesh3108/Url_Shortener.git
```

Step 2: cd into the cloned repository and run:

```bash
npm install
```

Step 3: Install & Run MongoDB (windows)

See <https://docs.mongodb.com/manual/installation/> for more informations

Step 4 (optional): If Nodemon not installed

```bash
npm install -g nodemon
```

Step 5: Start the application by

```bash
npm start
```
